## Getting Started

1 . load jquery
2 . load easyPlug
3 . load check
4 . you are ready to use check

```html
<script src="jquery.js"></script>
<script src="jquery.easyplug.js"></script>
<script src="jquery.check.js"></script>
<script>
jQuery(function($) {
  $('foo').check();
});
</script>
```
